## Опис проекту
У межах ЛР6 розроблено власну типізовану колекцію ArrayTaxiSet (на базі завдання ЛР5), що реалізує інтерфейс Set 
та використовує внутрішню структуру на основі масиву з початковою ємністю 15 елементів та динамічним розширенням 
на 30%.  
Колекція гарантує унікальність елементів та підтримує всі обов’язкові операції Set:
додавання, видалення, пошук, ітерацію, перевірку на наявність елементів тощо.  
Клас MainSetDemo запускає виконання Лабораторної роботи 6, демонструючи роботу власної реалізації множини 
ArrayTaxiSet.  
Проєкт структурований відповідно до принципів ООП, а реалізовані класи та методи забезпечують повноцінну модель 
роботи таксопарку та ілюструють застосування власних колекцій у Java.
